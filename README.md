# HermitChat

The companion mobile application for HermitShell, built with Flutter.

HermitChat allows you to manage your AI agents, converse with them, view system metrics, and interact with the generative Apps they create—all from your mobile device.

## Features

- **Agent Management**: View and interact with active HermitShell agents.
- **AI Chat Interface**: Real-time communication with agents with context visibility.
- **System Dashboard**: Monitor host metrics like CPU and Memory usage from the dashboard.
- **Apps Tab**: A built-in WebView interface to browse and use HTML/JS/CSS prototypes or apps generated by agents.
- **Settings Configuration**:
  - Dynamically toggle the execution of the Cloudflare Proxy Tunnel to expose the system externally.
  - Set customized LLM API tokens (OpenRouter, Anthropic, OpenAI) remotely.
  - Setup Timezone offsets to sync context for agents.

## Getting Started

1. Set up your [HermitShell Server](https://github.com/your-repo/HermitShell).
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```
4. Authenticate using your HermitShell server's IP/Hostname and the Administrator credentials (`admin` / `hermit123` by default).

## Requirements
- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
